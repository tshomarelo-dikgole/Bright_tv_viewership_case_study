This project showcase an end to end data analysis workflow in Databricks SQL.The goal was to transform raw TV viewership and userprofile data into meaningful insights that support audience analysis and reporting. 

The project demonstrates data cleaning,transformation, aggregation, and exploratory analysis using SQL in databricks where I used inner left join to join the two spreadsheets to gain insights.

Business objectives- Understand who is watching, what they are watching and when they are watching by analysing :
- User demographics
- Viewing behaviour
- Watch duration patterns
- Time- based engagement trends

  
This analysis helps Bright TV optmise programme schedules , improve targeted advertising and understand audience demographics.
